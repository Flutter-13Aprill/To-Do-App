## 💻 Setup Instructions

1- Clone the repository:

  ```
  git clone https://github.com/Saad-Alwazrah/To-Do-App.git
  ```

2- Navigate to the project directory:

   ```
   cd To-Do-App
   ```

3- Install dependencies:

   ```
   flutter pub get
   ```

4- Run the app:

   ```
   flutter run
   ```
